# Call Care tech test backend

## Things I learnt

How to develop a login system not using AWS Cognito in Flask.

# to install packages

```commandline
pip install -r requirements.txt
```


## To run code

```commandline
python3 run.py
```

## To run migration

```commandline
flask db migrate -m "commit message" 
```

## To apply migration
```commandline
flask db upgrade
```
